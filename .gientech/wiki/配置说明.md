# 配置说明

## 配置文件结构

```
company-rag-bootstrap/src/main/resources/
├── application.yml          # 公共配置（所有环境共享）
├── application-dev.yml      # 开发环境配置
└── application-prod.yml     # 生产环境配置
```

通过 `SPRING_PROFILES_ACTIVE` 环境变量切换环境，默认 `dev`。

## 公共配置 (application.yml)

### 服务器配置
- `server.port`：端口，默认 8080

### 数据源
```yaml
spring.datasource:
  url: jdbc:postgresql://${POSTGRES_HOST:localhost}:${POSTGRES_PORT:5433}/${POSTGRES_DB:company_rag}
  username: ${POSTGRES_USER:postgres}
  password: ${POSTGRES_PASSWORD:}
```

### PGVector 配置
```yaml
spring.vectorstore.pgvector:
  index-type: HNSW
  distance-type: COSINE_DISTANCE
  dimension: 1024
  remove-existing-vector-store-table: false
```

### Redis 配置
```yaml
spring.data.redis:
  host: ${REDIS_HOST:localhost}
  port: ${REDIS_PORT:6379}
  password: ${REDIS_PASSWORD:}
  database: 0
```

### MyBatis-Plus 配置
- 驼峰映射：开启
- 逻辑删除字段：`deleted`（0=未删除，1=已删除）
- 主键策略：auto

### Resilience4j 配置
```yaml
resilience4j:
  circuitbreaker:
    sliding-window-size: 10
    minimum-number-of-calls: 5
    failure-rate-threshold: 50
    wait-duration-in-open-state: 30s
  ratelimiter:
    limit-for-period: 10
    limit-refresh-period: 1s
    timeout-duration: 500ms
```

### Actuator / Prometheus
```yaml
management:
  endpoints:
    web:
      exposure:
        include: health,info,prometheus,metrics
```

### 日志配置
- `com.company.rag`：DEBUG
- `org.springframework.ai`：INFO

### 文件上传限制
- 单文件最大：50MB
- 请求最大：100MB

## 开发环境配置 (application-dev.yml)

### AI 模型配置
```yaml
spring.ai.openai:
  api-key: ${DASHSCOPE_API_KEY:default_dashscope_api_key}
  base-url: https://dashscope.aliyuncs.com/compatible-mode
  chat:
    options:
      model: deepseek-v4-flash
      temperature: 0.7
  embedding:
    api-key: ${SILICONFLOW_API_KEY:default_siliconflow_api_key}
    base-url: https://api.siliconflow.cn
    options:
      model: BAAI/bge-large-zh-v1.5
```

### 日志级别
- `com.company.rag`：DEBUG

## 环境变量

| 变量名 | 说明 | 默认值 |
|--------|------|--------|
| DASHSCOPE_API_KEY | 通义千问 API Key | 无 |
| SILICONFLOW_API_KEY | 硅基流动 API Key | 无 |
| POSTGRES_HOST | PostgreSQL 主机 | localhost |
| POSTGRES_PORT | PostgreSQL 端口 | 5433 |
| POSTGRES_DB | 数据库名 | company_rag |
| POSTGRES_USER | 数据库用户 | postgres |
| POSTGRES_PASSWORD | 数据库密码 | 空 |
| REDIS_HOST | Redis 主机 | localhost |
| REDIS_PORT | Redis 端口 | 6379 |
| REDIS_PASSWORD | Redis 密码 | 空 |
| SERVER_PORT | 服务端口 | 8080 |
| SPRING_PROFILES_ACTIVE | 环境 profile | dev |
